在github上搭建博客的思路和方法

GitHub pages提供了搭建免费博客的平台。

只要将搭建好的博客代码上传至repository即可。

因此，利用hexo可以搭建一套博客系统，具有编译和部署博客的功能。

hexo借助于git实现与GitHub的通信，完成上传博客的功能。
在git与GitHub通信之前需要先配置ssh公钥和私钥。

在安装hexo之前，需要安装git，node.js。

hexo运行在git bash命令行下，git bash在本机中的博客目录下启动，
通过命令来编译和部署。
